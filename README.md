# Gianni-Poulis

Overview

This project presents a data-driven asset management dashboard developed to evaluate the financial performance of a residential real estate portfolio. The objective was to identify revenue optimization opportunities, measure portfolio efficiency, assess tenant credit exposure, and support investment-oriented decision-making through interactive analytics.

Business Problem

Residential portfolios often contain unrealized income potential due to below-market rents, long-term tenancies, regulatory constraints, and operational inefficiencies. At the same time, portfolio managers must balance revenue growth with tenant risk, maintenance costs, and regulatory compliance.

The objective of this project was to design an executive dashboard that transforms operational portfolio data into actionable insights for asset management.


Business Objectives

The dashboard was designed to answer the following business questions:

How much annual revenue is currently generated by the portfolio?
Where is the greatest embedded revenue opportunity?
How efficiently are rents being captured relative to legal rent levels?
Which tenants represent the highest credit exposure?
How does property quality influence rental income?
What financial impact do renovation opportunities create?
Which units should be prioritized for revenue optimization?


Dataset

The project uses an anonymized residential portfolio dataset containing:

Unit information
Current and legal rents
Tenant tenure
Property type
Income segmentation
Maintenance costs
Payment history
Amenity information
Arrears balances

All personal and confidential information has been removed.

Methodology

The analysis was developed using Microsoft Excel and Power Query.

Key analytical techniques include:

Data transformation using Power Query
Pivot Table modelling
Revenue benchmarking
KPI development
Rent gap analysis
Tenant risk segmentation
Maintenance cost analysis
Executive dashboard design

Key Insights
Identified over €309K in embedded annual revenue upside.
Measured portfolio rent efficiency against legal rent benchmarks.
Quantified annual revenue exposed to high-risk tenants.
Demonstrated measurable rent premiums associated with renovated units.
Highlighted priority units for revenue optimization.
